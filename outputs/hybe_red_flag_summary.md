
# HYBE Audit Red Flag Agent

## 분석 개요
- 대상 기업: 하이브
- 재무제표 기준: 연결
- 분석기간: 2023년~2025년
- 금액 단위: 백만원
- 현재 분석 완료 계정 수: 8
- 현재 식별된 관계형 Red Flag 수: 3

## 핵심 위험 요약

## RF-001. 외형 성장과 수익성 악화의 괴리

- 상태: 추가 검토 필요
- 규칙 충족 여부: True
- 추가 확인 우선순위: 높음
- 근거 강도: 중간
- 데이터 완전성: 낮음
- 우선 검토 이유: 매출과 영업이익의 관계만으로 신호가 명확하게 탐지됐으나, 비용 세부내역과 사업부문별 자료가 아직 없어 원인을 확정할 수 없음

## RF-002. 이익과 영업현금흐름 방향의 불일치

- 상태: 손익과 현금흐름 구성요인 추가 검토
- 규칙 충족 여부: True
- 추가 확인 우선순위: 높음
- 근거 강도: 중간
- 데이터 완전성: 낮음
- 우선 검토 이유: 손익과 현금흐름의 방향 차이는 확인됐으나 조정항목과 운전자본 세부자료가 아직 없음

## RF-003. 높은 영업권 비중과 수익성 악화의 결합

- 상태: 영업권 및 관련 현금창출단위 손상평가 추가 검토
- 규칙 충족 여부: True
- 추가 확인 우선순위: 높음
- 근거 강도: 중간
- 데이터 완전성: 중간
- 우선 검토 이유: 영업권 비중과 영업이익률 하락은 확인됐지만, 현금창출단위별 손상검사 가정과 민감도 자료는 아직 확인하지 못함


## 분석 결과의 구분
- 확인된 사실: 입력된 재무데이터와 Python 계산 결과
- 분석 신호: 사전에 정의된 관계형 Red Flag 규칙
- 가능한 해석: 추가 확인이 필요한 가설
- 최종 판단: 사용자와 감사인의 전문가적 판단 필요

## 공통 주의사항
식별된 Red Flag는 오류나 부정의 존재를 의미하는 것이 아니라, 기업 및 환경에 대한 이해와 중요왜곡표시위험 평가를 위해 추가적인 질문과 검토가 필요한 영역을 의미합니다.
